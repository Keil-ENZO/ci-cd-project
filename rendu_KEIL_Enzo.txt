--------------------------------------------------
Note sur les tests et le coverage
--------------------------------------------------
Pour la couverture de tests, j'ai fait le choix d'exclure le dossier
"src/components/ui" du calcul Codecov. Comme j'ai utilisé la librairie
Shadcn UI pour le design, ce dossier contient uniquement des composants
visuels générés automatiquement : les tester n'avait pas de sens pour
l'exercice et faisait baisser le score pour rien. En revanche, tout le
code réellement écrit (logique du formulaire, pages, règles de validation,
authentification, composants admin) est couvert à plus de 95 %.
