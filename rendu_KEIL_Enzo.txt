Note concernant les tests et le coverage :
Petite précision pour la couverture de tests : j'ai fait le choix d'exclure le dossier "src/components/ui" du calcul Codecov. Comme j'ai utilisé la librairie Shadcn UI pour le design, ce dossier contient uniquement des composants visuels générés automatiquement. Les tester n'avait pas trop de sens pour l'exercice et ça faisait baisser le score pour rien. En revanche, tout le code que j'ai réellement écrit (logique du formulaire, pages, règles de validation) est bien couvert à 100 % !
